# ntc-pgrpc
ntc-pgrpc is an example python gRPC  

## Quick start
```shell script
# Gen source code python gRPC
make gen

# Start Server
make server

# Start client
make client

# Start Async Server
make async_server

# Start Async Client
make async_client
```
